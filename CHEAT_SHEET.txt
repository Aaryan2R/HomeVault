HomeVault 1-Page Cheat Sheet

Project Name:
HomeVault

What is it?
HomeVault is a local private cloud storage web app.
It lets users upload, store, search, filter, download, and delete files from a browser.

Main Goal:
To create a simple Google Drive or Google Photos like system for home use on a local computer.

Problem it solves:
- cloud storage gets full fast
- paid storage is costly
- manual backup on SSD/USB is inconvenient
- local storage is more private

Technology Used:
- Python Flask
- SQLite
- HTML
- CSS
- Jinja
- Pillow

Why these technologies?
- Flask handles the backend and routes
- SQLite stores metadata in one simple file
- HTML/CSS builds the interface
- Jinja shows Python data in HTML
- Pillow creates image thumbnails

Main Features:
- file upload
- auto file categorization
- file list
- image thumbnails
- search by filename
- folder/category filter
- download
- delete

Main Categories:
- Photos
- Videos
- Documents
- Others

Important Idea:
The real files are stored on disk.
The database stores only metadata.

What is metadata?
Metadata means file information such as:
- name
- size
- type
- folder
- upload date

Why not store files in the database?
Because storing files on disk and only metadata in the database is simpler, faster, and easier to manage.

Main Files:
app.py
- main backend logic
- handles routes

database.py
- database functions
- insert, fetch, search, delete

thumbnailer.py
- creates thumbnails for photos

templates/index.html
- user interface page

static/css/style.css
- page styling

How upload works:
1. user selects a file
2. browser sends it to Flask
3. Flask checks extension
4. Flask chooses category folder
5. Flask creates unique filename using UUID
6. file is saved in storage
7. metadata is saved in SQLite
8. thumbnail is created if it is a photo
9. homepage reloads

Why UUID is used:
UUID gives every uploaded file a unique saved name.
This prevents filename conflicts.

How search works:
1. user types in search box
2. browser sends GET request
3. Flask reads query
4. database finds matching files
5. results are shown

How filter works:
1. user clicks a category like Photos
2. browser sends folder name in URL
3. Flask reads folder value
4. database returns only matching files

How download works:
1. user clicks download
2. Flask finds file record by ID
3. Flask sends file back to browser

How delete works:
1. user clicks delete
2. Flask removes database record
3. Flask removes actual file from storage
4. Flask removes thumbnail if present

What is Flask?
Flask is a Python web framework used to build the backend.

What is SQLite?
SQLite is a lightweight file-based database.

What is Jinja?
Jinja is a template engine that shows Python data inside HTML.

What is Pillow?
Pillow is a Python library used for image processing and thumbnails.

Short Viva Answer:
HomeVault is a local private cloud storage web application built with Flask and SQLite. It stores uploaded files in categorized folders on the local system and stores file metadata in a database. It supports upload, search, filter, download, delete, and image thumbnail generation.

Future Scope:

- trash/recycle bin
- better preview system
- dashboard/stats
