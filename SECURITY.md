|---------|-----------|

## Reporting a vulnerability

HomeVault is designed for local network use only and is not intended to be exposed to the public internet.

If you find a security issue, please do not open a public GitHub issue. Instead:

1. Email aaryan08072005@gmail.com with the subject: `[HomeVault Security]`
2. Describe the vulnerability and steps to reproduce
3. I will respond within 7 days

Thank you for helping keep HomeVault secure.
